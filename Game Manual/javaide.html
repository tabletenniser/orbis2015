Setting up your IDE¶

An IDE can help with coding. 3 IDEs are demonstrated: Eclipse (recommended), IntelliJ, and BlueJ.

Python¶

To start the game, navigate to your development kit Folder and open the 3 batch file on Windows or 3 shell files on linux in the order given below. You can also call these in command line and pass parameters through them. Write your AI in PlayerAI.py

Eclipse (recommended)¶

Set Up¶

  1. Set up a workspace.
  1. Set up a new project under File -> New -> Java Project.
  1. Name your project and hit next.
  1. Check on Allow output folder for source folders then hit Libraries.
  1. Click Add External JARs.
  1. Navigate to the Java Library Folder and add both libraries.
  1. Close the Welcome Menu.
  1. Copy all the files in the Java folder into the project.
  1. Drag PlayerAI.java from botplayer into src. Your eclipse should look like the below picture.
  1. Start coding in PlayerAI.java

Starting Game¶

  1. To start a game, right click BotRampage.jar under Referenced Libraries. Go to Run As and click Java Application.
  1. Select RunGame - com.orbischallenge.main as the main class to run. A window will pop up.
  1. Right click RunClient.jar under Referenced Libraries. Go to Run As and click Java Application.

Next Steps: If you need command line options: go to Run Configurations and change arguments to RunGame.

IntelliJ¶

Set Up¶

  1. Click Create New Project.
  1. Select JDK if needed by clicking New... -> JDK and select the appropriate JDK.
  1. One IntelliJ has detected JDK, click Next.
  1. Name your project and hit Finish.
  1. Copy all the files in the folder ‘Java’ into your project (top level folder) except PlayerAI.class if it exists. Drag and drop would work as well as copy and pasting.
  1. Right click the folder botplayer and go to Mark Directory As and click Sources Root.
  1. Go to Java Library and right click BotRampage.jar and click Add as Library.
  1. Go to Java Library and right click JavaClient.jar and click Add as Library.
  1. Start coding in PlayerAI.java.

Starting Game¶

  1. To start the game, navigate to your IntelliJ Project Folder and open the 3 batch file on Windows or 3 shell files on linux in the order given below. You can also call these in command line and pass parameters through them.

BlueJ¶

Set Up¶

  1. Go to Project and click Open Non BlueJ.
  1. Navigate and select the folder botplayer and click open in BlueJ.
  1. Go to Tools and select Preferences.
  1. Go to the Libraries tab and click Add.

5.Open BotRampage.jar.

  1. Do the same with JavaClient.jar.
  1. Click Ok on both windows.
  1. Click Tools and Click on Rebuild Package.
  1. Click Tools and Click on Reset Java Virtual Machine.
  1. Start coding in PlayerAI.java.
